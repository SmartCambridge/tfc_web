Description of parking data

Data

One row for each observation for each car park with these columns:

Column name Description
parking_id Unique identifier for the car park.
ts Observation timestamp in seconds since 00:00:00 UTC on 1 January 1970, minus leap seconds ('Unix' or 'Epoch' time).
ts_text Observation timestamp as text in UK local time (UTC in the winter, BST in the summer) with timezone information.
spaces_capacity The total number of spaces in the car park at the time of observation.
spaces_occupied The number of spaces in use in the car park at the time of observation.
spaces_free The number of available spaces in the car park at the time of observation.

'spaces_occupied' + 'spaces_free' will always equal 'spaces_capacity'.

Metadata

One row for each car park with these columns:

Column name Description
parking_id Unique identifier for the car park
parking_name Human-readable name for the car park
parking_type Type of car park ('car_park', 'park_and_ride')
latitude WGS84 latitude of approximate centre of the car park
longitude WGS84 longitude of approximate centre of the car park

There's a map showing all currently monitored car parks.
