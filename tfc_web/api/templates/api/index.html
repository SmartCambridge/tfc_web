Smart Cambridge APIs

Programmatic access to information maintained by the Smart Cambridge project.

Currently available:

  • Air Quality data from a selection of sensors deployed around Cambridge between June 2016 and February 2017.
  • Car Parking data showing the occupancy of Cambridge City Centre and Park and Ride car parks from 2017 onward.
  • Traffic Speed data for a number of roads within Cambridge based on bus position reports from October 2017 onward.
  • Bus transport data from the National Public Transport Access Nodes (NaPTAN) and Traveline National Dataset (TNDS) resources.

Access to all endpoints requires free registration and authentication. Both session authentication (by logging in to SmartCambridge) and API Token authentication are supported. API tokens are created and managed through pages linked below. For clients to authenticate using a token it must be included in an Authorization HTTP header. The token should be prefixed by the string literal "Token", with whitespace separating the two strings. For example:

Authorization: Token xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx

Access to all endpoints is also rate-limited, currently to 1200 requests per minute and 12000 requests per hour per token (for token-based access) or per user (for session authentication).
