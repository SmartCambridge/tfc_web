Smart Cambridge Data API and data downloads

SmartCambridge's work is focused on acquiring, processing and using data in real time. As part of that processing we archive the data we collect and make it available for reuse.

Archived data can be accessed in two ways:

  • By Download which provide the data in bulk as 'comma separated variable' (CSV) files which can be imported into a range of tools for analysis.
  • Via the Program API which is ideal for use by programs and apps.

Available data:

  • Air Quality data from a selection of sensors deployed around Cambridge between June 2016 and February 2017.
  • Car Parking data showing the occupancy of Cambridge City Centre and Park and Ride car parks from 2017 onward.
  • Traffic Speed data for roads within Cambridge based on bus position reports from October 2017 onward.
  • (Via the Program API only) bus transport data from the National Public Transport Access Nodes (NaPTAN) and Traveline National Dataset (TNDS) resources.

The data is available to anyone, but you have to register and agree to the platform's Terms and Conditions before you can access it.
