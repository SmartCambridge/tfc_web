Description of air quality data

Data

Air quality data is provided seperate 'data' and 'header' files:

'Data' files

One row representing each observation from each sensor at each station location, with these columns:

Column name Description
station_id Unique identifier of the sensor station.
sensor_type The type of sensor.
ts Observation timestamp in miliseconds since 00:00:00 UTC on 1 January 1970, minus leap seconds.
ts_text Observation timestamp as text in UK local time (UTC in the winter, BST in the summer) with time zone information.
reading Sensor reading (units depending on sensor type).

'Header' files

One row per month for each sensor at each station location, with these columns:

Column name Description
station_id Unique identifier of the sensor station.
sensor_type The type of sensor.
ts Observation timestamp in miliseconds since 00:00:00 UTC on 1 January 1970, minus leap seconds.
ts_text Observation timestamp as text in UK local time (UTC in the winter, BST in the summer) with time zone information.

together with some or all of the folowing columns: 'BatteryVoltage', 'COFinal', 'COOffset', 'COPrescaled', 'COScaled', 'COSerialNumber', 'COSlope', 'COStatus', 'GasProtocol', 'GasStatus', 'Humidity', 'Latitude', 'Longitude', 'Name', 'NO2Final', 'NO2Offset', 'NO2Prescaled', 'NO2Scaled', 'NO2SerialNumber', 'NO2Slope', 'NO2Status', 'NOFinal', 'NOOffset', 'NOPrescaled', 'NOScaled', 'NOSerialNumber', 'NOSlope', 'NOStatus', 'O3Final', 'O3Offset', 'O3Prescaled', 'O3Scaled', 'O3SerialNumber', 'O3Slope', 'O3Status', 'OtherInfo', 'P1', 'P2', 'P3', 'ParticleNumber', 'ParticleProtocol', 'ParticleStatus', 'PM10Final', 'PM10Offset', 'PM10Output', 'PM10PreScaled', 'PM10Slope', 'PM1Final', 'PM1Offset', 'PM1Output', 'PM1PreScaled', 'PM1Slope', 'PM2_5Final', 'PM2_5Offset', 'PM2_5OutPut', 'PM2_5PreScaled', 'PM2_5Slope', 'PMTotalOffset', 'PMTotalPreScaled', 'PMTotalSlope', 'PodFeaturetype', 'Pressure', 'SerialNo', 'SO2Final', 'SO2Offset', 'SO2Prescaled', 'SO2Scaled', 'SO2SerialNumber', 'SO2Slope', 'SO2Status', 'T1', 'T2', 'T3', 'Temperature', 'TSP'.

Metadata

One row for each station location with these columns:

Column name Description
station_id Unique identifier of the sensor station.
Name Human-readable name of the sensor station.
Description Human-readable description of the sensor station.
SensorType A list separated by '|' of the types of sensor available at this sensor station.
Latitude Latitude of the sensor station.
Longitude Longitude of the sensor station.
date_from The earliest date for which data is available for this station.
date_to The latest date for which data is available for this station.

There's a map showing all the sensor locations.
