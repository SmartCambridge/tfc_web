Description of bluetooth-based traffic speed data

Data

One row for each observation of traffic speeds along monitored section of road with the following columns:

Column name Description
id Identifier of the section of road involved.
ts Timestamp in seconds since 00:00:00 UTC on 1 January 1970, minus leap seconds ('Unix' or 'Epoch' time) of the retrieval of this record.
ts_text Timestamp as text in UK local time (UTC in the winter, BST in the summer) with timezone information of the retrieval of this record.
time Time in text format of the most recent journey time calculation for this record.
period The time in seconds over which journeys were observed to calculate travelTime.
travelTime The time in seconds taken by traffic transiting the section of road.
normalTravelTime A long term average travelTime for this section of road based on the time of day, day of week, Bank Holidays and school terms.

Link Metadata

One row for each monitored 'link' between a pair of sensor sites, with the following columns:

Column name Description
id Link identifier.
ts Timestamp in seconds since 00:00:00 UTC on 1 January 1970, minus leap seconds ('Unix' or 'Epoch' time) of the retrieval of this record.
ts_text Timestamp as text in UK local time (UTC in the winter, BST in the summer) with timezone information of the retrieval of this record.
name Name for this link.
description Description of this link.
sites A list separated by '|' of the identifiers of the two sensor sites defining the link.
length The length in meters of this link.

Route Metadata

One row for each monitored 'compound route' built from individual links, with the folowing columns:

Column name Description
id Route identifier.
ts Timestamp in seconds since 00:00:00 UTC on 1 January 1970, minus leap seconds ('Unix' or 'Epoch' time) of the retrieval of this record.
ts_text Timestamp as text in UK local time (UTC in the winter, BST in the summer) with timezone information of the retrieval of this record.
name Name for this route.
description Description of this route.
sites A list separated by '|' of the identifiers of the the sensor sites defining the route.
links A list separated by '|' of the identifiers of the links making up the route.
length The length in meters of the route.

Site Metadata

One row for each monitoring station with the following columns:

Column name Description
id Station identifier.
ts Timestamp in seconds since 00:00:00 UTC on 1 January 1970, minus leap seconds ('Unix' or 'Epoch' time) of the retrieval of this record.
ts_text Timestamp as text in UK local time (UTC in the winter, BST in the summer) with timezone information of the retrieval of this record.
name Station name.
description Station description.
latitude Station latitude.
longitude Staion longitude.
