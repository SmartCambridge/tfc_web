Create a DISPLAY if you want to keep the SmartPanel open

The 'layout' runtime limit is capped (currently at 10 minutes).

Please note these quick 'layout' views are intended for checking your work-in-progress as you design a great SmartPanel. Layouts left open in a browser unnecessarily consume resources, some of which have directly associated costs for us.

If you want to view a SmartPanel for more than a few minutes, please go back into the SmartPanel design menu and create a new DISPLAY, and choose your existing layout for that display. The display will be given a unique identifier, such as 'ABCD-1234', and then you can access that SmartPanel permanently at /smartpanel/display/<your unique identifier>.

To create and configure a new display, go to your displays list page.
